An experimental Redis Ruby client.

## Compatibility

- Ruby 2.2.2+
- JRuby 9k+

## Supported features

- [Pipeling](http://www.rubydoc.info/github/etehtsea/oxblood/master/Oxblood/Pipeline)
- [Connection pooling](http://www.rubydoc.info/github/etehtsea/oxblood/master/Oxblood/Pool)
